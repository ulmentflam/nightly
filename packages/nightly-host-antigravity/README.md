# nightly-host-antigravity

[Nightly](../../README.md) host integration for **Google Antigravity** —
secondary host. Registers as a Managed Agent in the Agent Manager. Mirrors
Nightly's per-task layout into Antigravity's `~/.gemini/antigravity/brain/<GUID>/`
so the Agent Manager UI shows familiar walkthroughs.

**Status:** Phase 0 stub. Real implementation lands in Phase 6.
