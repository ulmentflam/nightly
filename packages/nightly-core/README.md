# nightly-core

The host-agnostic core of [Nightly](../../README.md): loop, priority cascade,
drain protocol, atlas builder, briefing renderer. Per-host integrations (Claude
Code, Codex, Cursor, opencode, Antigravity) plug into this package.

**Status:** Phase 0 skeleton. See [`.planning/brainstorm.html`](../../.planning/brainstorm.html)
for the design.
